# 关于 HYJ

你好，欢迎来到我的博客空间。

我是 HYJ，一个对技术和工程抱有浓厚兴趣的开发者。喜欢在终端和代码之外，把想法变成能跑起来的东西。这里主要用来记录我的编程学习过程，以及一些折腾时的笔记和思考。

**目前主要关注：**
- 后端开发与系统设计
- Minecraft 模组与生电工程
- 开源工具与自动化流程
- 自行车与物理机械
- 函数与数学直觉

这个博客用 [Gmeek](https://github.com/Meekdai/Gmeek) 驱动，所有的文章都通过 GitHub Issues 管理，自动构建，简单可靠。我不太擅长做长篇大论的输出，但会尽量把一些值得记录的点写下来。

如果你对某个项目或想法感兴趣，欢迎在对应文章评论区讨论，或者直接通过 GitHub 联系我。觉得有意思的点个 Star，觉得有问题的提个 Issue。

感谢你的阅读。
